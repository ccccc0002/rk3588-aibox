# RK3588 AI Box

This repository hosts the RK3588 AI box project for IPC analytics, preview relay, protocol integration, and cockpit UI.

## Current Scope (V1)
- 20 IPC channels
- Main-stream preview relay (no server-side transcoding)
- Adaptive frame sampling analysis
- GAT1400 bidirectional integration
- GB28181 access and cascade
- RBAC with tenant/project isolation
